# Input Component Style Guide

This folder documents the design principles, interactions, and visual characteristics for each supported input style in the Input component.

## Style Categories

### Classic Design Systems

- **Material Design** — Google's design language with floating labels, ripple effects, elevation
- **Material You (M3)** — Material 3 with dynamic color, rounded containers, tonal elevation
- **Fluent UI** — Microsoft's design with acrylic, reveal effects, bottom accent borders
- **iOS Modern** — Apple's HIG with SF-style rounded rects, subtle shadows, clean typography

### Flat & Minimal

- **Flat Design** — No shadows, no gradients, solid colors, clean edges
- **Flat 2.0** — Flat with subtle shadows and depth cues for usability
- **Minimal UI** — Stripped to essentials, underline-only, maximum whitespace

### Glass & Transparency

- **Glassmorphism** — Frosted glass, backdrop blur, transparency, light borders
- **Liquid Glass** — Apple's iOS 26 style, refractive glass, dynamic highlights
- **Acrylic UI** — Microsoft's acrylic material, noise texture, layered blur
- **Frosted Glass UI** — Softer glass effect, muted blur, ethereal feel

### Depth & Shadow

- **Neumorphism** — Soft UI, extruded/inset shadows, monochromatic, plastic feel
- **Gradient UI** — Gradient backgrounds, smooth color transitions

### Bold & Expressive

- **Brutalist UI** — Raw, harsh borders, offset shadows, monospace, no rounded corners
- **Memphis UI** — 80s postmodern, multi-colored offset shadows, geometric patterns
- **High-Contrast UI** — Maximum contrast, thick borders, accessibility-first

### Retro & Nostalgic

- **Retro UI** — 70s/80s computing, inset bevels, warm tones
- **Vintage UI** — Aged paper, serif fonts, subtle textures
- **Pixel UI** — 8-bit gaming, chunky borders, pixel fonts
- **Y2K UI** — Early 2000s, bubbly gradients, rounded, playful colors
- **Vaporwave UI** — 80s/90s nostalgia, purple/pink/cyan, letter-spacing, surreal

### Futuristic & Sci-Fi

- **Cyberpunk UI** — Neon glow, glitch effects, clipped corners, dark backgrounds
- **Neon UI** — Glowing borders, dark background, light bloom effects
- **Sci-Fi UI** — HUD-style, monospace, uppercase, scan-line feel
- **Futuristic UI** — Smooth gradients, glass-like, subtle glow, rounded
- **Terminal UI** — Green-on-black, monospace, command-line aesthetic

### Theme Variants

- **Monochrome UI** — Black and white only, clean and stark
- **Dark Theme UI** — Dark surfaces, light text, subtle borders
- **Light Theme UI** — White surfaces, subtle shadows, clean borders
